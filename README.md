# Instructions
1. Build a photo booth (see below)
2. Connect your Pi and PiCamera
3. Connect a button to the Pi's GPIO21 and Ground pins.
4. Download the code
5. Run:
`python camera.py`
5. Photos will get saved to photos directory where you can elect to publish them later.
